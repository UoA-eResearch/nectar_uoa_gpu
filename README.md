# nectar_uoa_gpu

Prints a list of UoA GPUs and the projects currently attached to them

```
usage: gpu.py [-h] [-u EMAIL] [-s] [-l]

Nectar GPU script

optional arguments:
  -h, --help              show this help message and exit
  -u EMAIL, --user EMAIL  print just this users project(s)
  -s, --short             long format
  -l, --long              long format
```
